MessageBuilder & IRC::MessageBuilder::operator<< ( const std::string &  arg)

Inserts an argument at the end of the list.

If the argument is empty and is not the last one, it will be replaced by '*'. Note that only the last argument will have ':'. Any arguments before the last containing ' ' or ':' result in undefined behavior.

◆ str()

std::string IRC::MessageBuilder::str ( ) const

Convert the message into a string compatible with the IRC protocol which is :prefix arg*(' 'arg) [' ':last_arg].

All empty arguments are replaced by an asterisk ('*').

Returns
The string using the format required by the IRC protocol without newline
