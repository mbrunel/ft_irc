Checks if a string matches a mask.

Supported operators are match one (?), match many or wilcard (*) and escape (\).

Examples:
  • match("a*", "abc") -> true
  • match("a*", "bcd") -> false
  • match("x?z", "xyz") -> true
  • match("\*", "*") -> true
Parameters
maskThe mask used for matching
strThe string to match
Returns
true if a match is found
